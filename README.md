# Unagi

An RPG editor on modern browsers


# Environment

This is tested only on Mac OS X Chrome so far.


# How to run

## Requirements

* [Go](http://golang.org/)
* [TypeScript](http://www.typescriptlang.org/)

## Steps

1. Run `./build.sh`
2. Run `go run testserver.go`
3. Access to http://localhost:8787/
